# Design System Documentation

## Overview
This design system outlines the visual and structural guidelines used in building the my resume page. It includes fonts, colors, and component layout.

---

## Color Palette

- **Primary Color:** #2980b9 (Blue)
- **Text Color:** #333 (Dark Grey)
- **Background Color:** #f4f4f4 (Light Grey)
- **Accent Color:** #2c3e50 (Dark Blue)
- **Links:** #2980b9 (Blue for links)


## Typography

- **Font for Body:** Arial, sans-serif
- **Font for Headers:** Arial, sans-serif

The use of a clean sans-serif font like Arial ensures readability across various devices. The larger font size for headers (3em) makes them stand out, while the 1.1em font size for body text is optimized for legibility.

## Components

### Header
The header includes the name, contact details, and has a central alignment with a background color of white for contrast.

- **Header Background:** White (#fff)
- **Header Text Color:** #2c3e50
- **Font Size:** 3em for the name

## Layout
The layout uses a flexible box layout (`flexbox`) to ensure the page is responsive on various screen sizes. Margins and padding are kept minimal to maintain a clean and simple design.
